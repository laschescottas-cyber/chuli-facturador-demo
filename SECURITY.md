# Seguridad

Este repositorio es una demostración pública y no incluye datos fiscales ni credenciales reales.

## Archivos que nunca deben subirse

- `.env` y sus variantes locales.
- Certificados y claves privadas (`.crt`, `.key`, `.pem`, `.p12`, `.pfx`).
- Bases de datos SQLite.
- Archivos CSV o Excel con ventas o información de clientes.
- Logs y copias de seguridad.
- CUIT, tokens o credenciales reales escritos en el código.

El archivo `.env.example` contiene únicamente valores de referencia y debe conservarse sin información real.

## Reporte de seguridad

Si encontrás una vulnerabilidad, no publiques credenciales ni datos sensibles en un Issue. Contactá a la autora mediante los datos disponibles en su perfil de GitHub.
